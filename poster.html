Problem Statement
Adversarial Search in a Two-Player Zero-Sum Game
Connect 4 is a classic toy problem: two players drop discs into a 7-column grid; first to place 4 in a row wins. For an AI this is a fully observable, deterministic, adversarial search problem requiring look-ahead reasoning.
State space: 6×7 board = 4.5 trillion positions
Branching factor: up to 7 (non-full columns)
Operators: drop disc into any valid column
Goal: 4 consecutive discs of same colour
Agents: MAX (AI) vs MIN (player) — zero-sum

Algorithm
Minimax
function minimax(board, depth, α, β, isMax): if depth == 0 or terminal: return heuristic(board) if isMax: score = −∞ for move in validMoves: score = max(score, minimax(...false)) α = max(α, score) if β ≤ α: break // prune else: score = +∞ for move in validMoves: score = min(score, minimax(...true)) β = min(β, score) if β ≤ α: break // prune
